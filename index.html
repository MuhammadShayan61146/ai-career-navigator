About AI Career Navigator

Mission, design philosophy, technology, and future direction.

🧭

AI Career Navigator

v1.0 — Stable Release Built by Muhammad Shayan Released July 2026

A free, privacy-first, single-file web application that gives every learner a structured, personalised path into AI — regardless of background, budget, or technical level. No login. No subscription. No tracking.

🎯 Mission

Millions of people want to learn AI but receive generic, one-size-fits-all curricula that ignore their background, goals, and available time. A marketing manager, a CS student, and a high school teacher should never follow the same learning path.

AI Career Navigator solves this by matching each learner to a curated 12-week roadmap tailored to their specific profile — and optionally using Gemini AI to write a fully custom curriculum from scratch.

🧩 Problem Being Solved
❌Generic AI courses ignore learner background and goals
❌Most learning tools fabricate progress metrics and skills
❌AI-generated career materials often invent experience
❌No free, offline-capable structured AI curriculum exists
✅This tool: curated paths, honest progress, evidence-based career documents only
⚡ Key Features
🧭
Curated Learning Paths
13 expert-designed 12-week roadmaps matched by background, goal, and time. Optional Gemini AI upgrade for a fully custom written roadmap.
🌳
Interactive Skill Tree
SVG visual map of learning progression. Nodes unlock and glow as weeks complete.
📊
Honest Progress Tracking
Streak calendar, weekly completion, hours studied, and achievements — all from real verified actions.
✅
Evidence-Based Career Docs
Resume bullets, LinkedIn summaries, GitHub READMEs — generated only from verified completed weeks. Nothing invented.
🤖
AI Mentor
Built-in AI concept explainer. Optional Gemini upgrade for contextual, multi-turn guidance aware of your roadmap.
💾
localStorage-Powered
All progress data lives in your browser's localStorage. No account, no server. Works without a network connection once the page is loaded — CDN resources (fonts, icons) require internet on first visit.
🛠 Technology Stack
FrontendVanilla HTML5, CSS3, JavaScript ES2022+ — no framework, no build step
FontsSpace Grotesk · Inter · JetBrains Mono via Google Fonts
IconsFont Awesome 6.5 via CDN
AI (opt.)Google Gemini 2.0 Flash — roadmap generation and AI Mentor
Newsrss2json.com free API — CORS-safe RSS → JSON conversion
GitHubGitHub REST API (public, unauthenticated)
StoragelocalStorage — browser-native, zero server required
🏗 Architecture & Design Philosophy
Single HTML file — Zero dependencies, zero build step, zero server. Open the file and it works.
Privacy-first — No user data leaves the browser. API keys go directly to Google's API — never to an intermediary.
Graceful degradation — Every feature works offline. AI features are optional enhancements, not requirements.
Honest by design — Career documents are generated exclusively from verified completed work. The system has no mechanism to invent experience.
Premium aesthetic — Design language inspired by Linear, Notion, Vercel, and Stripe.
