## 📌 What Is This?

AI Career Navigator is a single-file web application that gives every learner
a structured, personalised path into AI — regardless of background, budget,
or technical level. No login. No subscription. No tracking.

## ✨ Key Features

- **13 Expert-Curated Roadmaps** — matched by background, goal, and daily time
- **Interactive Skill Tree** — SVG-based visual progression that unlocks as you learn
- **AI Mentor** — built-in knowledge base; upgraded with Gemini AI when a key is provided
- **Evidence-Based Career Docs** — resume bullets and LinkedIn summaries from verified completed work only
- **Live AI News Feed** — from OpenAI, Anthropic, Google DeepMind, Hugging Face
- **GitHub Integration** — connect your username to see repos and project recommendations
- **Offline-First** — all data in localStorage, works with no internet after first load
- **Accessible** — skip links, ARIA roles, keyboard navigation, reduced-motion support
